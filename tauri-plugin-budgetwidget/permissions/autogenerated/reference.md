## Default Permission

Allows publishing the abstract budget-health ring fraction to the home-screen widget.

#### This default permission set includes the following:

- `allow-publish-health`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`budgetwidget:allow-publish-health`

</td>
<td>

Enables the publish_health command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`budgetwidget:deny-publish-health`

</td>

</td>
</tr>
