## Default Permission

Allows setting the system bar appearance to match the app theme.

#### This default permission set includes the following:

- `allow-set-dark`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`statusbar:allow-set-dark`

</td>
<td>

Enables the set_dark command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`statusbar:deny-set-dark`

</td>
<td>

Denies the set_dark command without any pre-configured scope.

</td>
</tr>
</table>
